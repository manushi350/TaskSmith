🛠️ TaskSmith – Forging Home Solutions

TaskSmith is a home services booking application that connects users with trusted service providers for everyday household tasks such as cleaning, cooking, gardening, childcare, and more.
The platform simplifies the process of finding help at home by allowing users to post tasks and service providers to accept and complete them.

This project aims to make home service management simple, fast, and reliable through an easy-to-use mobile interface and a scalable backend system.

🚀 Features

👤 Dual User System

Customer Login

Service Provider Login

📌 Post & Manage Tasks

Users can create tasks for services like cleaning, cooking, childcare, etc.

🤝 Task Acceptance

Service providers can browse and accept available tasks.

📍 Location-Based Services

Helps connect users with nearby service providers.

⭐ Ratings & Feedback

Users can rate service providers after task completion.

💳 Future Payment Integration

Secure payment support planned for future updates.

📱 Application Workflow

User registers or logs in to the app.

User posts a household task (cleaning, cooking, gardening, etc.).

Nearby service providers can view available tasks.

A provider accepts the task.

Task is completed and feedback can be given.
